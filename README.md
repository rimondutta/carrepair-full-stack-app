# Care Plus Auto Repairing 🚗✨

A premium, production-ready automobile maintenance and repair workshop application built with **Next.js 16**, **React 19**, and **Tailwind CSS**.

## 🏎️ Features

-   **High-End Design**: A stunning, dark-themed interface designed to wow users and feel premium.
-   **Dynamic Service Pages**: Individual pages for each service (Engine Tuning, Glass Repair, Car Wash, etc.) with detailed content.
-   **Production Ready**:
    -   **Optimized Metadata**: OpenGraph, Twitter cards, and title templates for superior social sharing.
    -   **SEO Excellence**: Dynamically generated `sitemap.xml` and `robots.txt`.
    -   **Error Handling**: Custom, branded 404 (Not Found) and global Error Boundary pages.
    -   **Performance**: Utilizes Next.js App Router and optimized fonts (Bai Jamjuree & Montserrat).
-   **Rich Content**: Expanded text-to-HTML ratios to improve search engine rankings.
-   **Interactive FAQ**: Service-specific FAQs to improve user engagement and SEO.

## 🛠️ Tech Stack

-   **Framework**: [Next.js 16 (App Router)](https://nextjs.org/)
-   **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
-   **Icons**: [Lucide React](https://lucide.dev/)
-   **Fonts**: [Google Fonts (next/font)](https://nextjs.org/docs/app/building-your-application/optimizing/fonts)
-   **Language**: [TypeScript](https://www.typescriptlang.org/)

## 🚀 Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📦 Deployment

To create an optimized production build:

```bash
npm run build
```

This project is optimized for deployment on the [Vercel Platform](https://vercel.com/new).
